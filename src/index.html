Freeze apps that are running in the background.


App by @actuallymentor

Apps to freeze:

Panic button
Toggle help

The text area accepts one process name pattern per line (regex compatible). For example "Google Chrome.*" would match all "Google Chrome" but also "Google Chrome Helper" processes.

The panic putton unfreezes everything on your system.

To see the code behind this app visit https://github.com/actuallymentor/mac-freeze-background/.
